O que é o HTML?

voltar

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web, nele é definido o significado e a estrutura do conteúdo de uma página Web, porém, para que a página HTML fique com a cara que vemos hoje, ele atua juntamente tanto com o CSS (Folhas de Estilo em Cascata), para a aparência e apresentação e o JavaScript para definir o comportamento e as funcionalidades do site.

A estrutura básica de um HTML

voltar

O HTML possui como estrutura base a seguinte sequência:

                 <!DOCTYPE html> 


                 <html> 

                  <head> 

                   <title> Nome ou título da página  </title> 

                   <meta charset="UTF-8"> 

                  </head> 

                  <body> 

                   Aqui dentro vai todo o conteúdo HTML do site 

                  </body> 

                </html>

O doctype não é uma tag HTML, mas uma instrução para o navegador e outros programas que podem ler seu site, que o código encontrado ali é um código HTML. Assim eles sabem o que fazer para mostrar seu site da melhor forma possível.

O doctype é OBRIGATÓRIO e deve ser sempre a PRIMEIRA LINHA do seu documento.

A tag head Contém informações que não são transpostas visivelmente para o usuário/leitor do documento. São dados implícitos, de uso e controle do documento: vinculação com outros arquivos, aplicação de lógica de programação de scripts e metadados. Na prática, todo o conteúdo do cabeçalho fica delimitado entre a abertura e fechamento tag head.

A tag body trata-se doa informação legível para o usuário/leitor do documento. É todo e qualquer texto que se deseja apresentar, assim como toda e qualquer forma de mídia de saída (imagens, sons, miniaplicativos embutidos, conteúdo multimídia, etc). Além disso, toda a apresentação de entrada de dados (formulários) também se aplica neste seção do documento.

O que são Tags

voltar

Tags são o conjunto de caracteres que formam um elemento, ou seja, quando nos referenciamos à Tag "p" estamos falando disso: </html>.

Existem dois tipos de Tags, as que necessitam de fechamento e as que não necessitam de fechamento.
Para as que necessitam de fechamento, utilizamos o sinal de menor ( < ), seguido do nome do elemento e o sinal de maior (> ) para abertura. Para fechamento, utilizamos o sinal de menor ( < ), seguido de barra ( / ), nome do elemento e o sinal de maior (> ).

Os elementos que não necessitam de fechamento, também conhecidos como elementos vazios, somente utilizamos o sinal de menor ( < ), seguido do nome do elemento e o sinal de maior (> ).


Os Atributos básicos

voltar

Atributos são informações que passamos na Tag para que ela se comporte da maneira esperada. Existem atributos globais (que funcionam em todas as Tags) e específicos (que são direcionados para cada Tag, através de especificação).

Os Atributos possuem nome e um valor, existem Atributos que você vai usar praticamente sempre e existem outros que serão mais raros.

Alguns atributos globais são:

  • class: Determina uma (ou mais) classe(s) para o elemento. Para definir mais de uma classe, coloque-as separadas por espaço.

  • id: É o identificador único do elemento. Somente deve ser declarado um id por elemento. E este id não deve ser repetido na mesma página.

  • style: Determina propriedades CSS diretamente no elemento.

  • title: Representa um auxílio ao elemento. Semelhante a dica do elemento.

Trabalhando com textos, imagens e audios no HTML

voltar

Para trabalhar com textos, em uma pagina HTML, existem várias tags que auxiliam na organização e na estruturação do texto. Vamos mostrar as principais tags.

Tags para títulos

Para trabalhar com títulos em uma página HTML utilizamos as tags h1 à h6, sendo que a tag h1 a de maior peso e a tag h6 de menor peso.

Tags para textos

Para trabalhar com textos em sí existem várias tags, algumas delas são:

  • p: Principal tag de texto, compõe um parágrafo
  • span: Apesar de ter uma funcionalidade e características parecidas com os parágrafos, costumam ser utilizadas apenas para pequenas informações, como legendas de um formulário, legendas de uma imagem, etc. Também pode ser utilizada para formar um container
  • pre: Tag utilizada para representar texto pré-formatado. Muito utilizada para inserir códigos
  • b: Transforma o conteúdo em negrito
  • strong: Transforma o conteúdo em negrito
  • i: Transforma o conteúdo em itálico
  • br: Essa tag não necessita de fechamento, ela executa a função de quebra de linha.
  • hr: Essa tag não necessita de fechamento, ela forma uma linha horizontal.

Tags para links

Para se inserir um link na página HTML, ou seja, fazer uma chamada para outro documento ou trechos específicos do mesmo documento, utiliza-se a tag a.

A estrutura de programação para uma tag a é:

<a href="https://www.google.com.br"> google <\a>

Gerando como resultado o link para a página do google.

Tags para imagens, vídeos e áudios

Para se trabalhar com imagens no HTML utilizamos a tag img e inserimos no atributo src o caminho para a imagem desejada e no atributo alt a descrição da imagem, caso ela não carregue na página HTML ou alguém com deficiência visual abra a página. Seguindo o exemplo:

<img src="https://cdn.icon-icons.com/icons2/17/PNG/256/social_html5_1970.png" alt="logo HTML5">

Para trabalharmos com vídeo utilizamos a tag video onde o principal atributo dela é o control. E para adicionarmos o vídeo dentro desta tag, utilizamos uma outra tag a tag source, onde nela será iserido o atributo src que terá o camigo do vídeo e o atributo type que será o tipo de vídeo que será inserido. Segue exemplo:

                    <video controls>
                        <source src="video.mp4" type="video/mp4">
                        <source src="video.webm" type="video/webm">
                        Seu navegador não possui suporte para Vídeos.
                    </video>

Já para trabalhar com áudio, utilizamos a mesma sintáxe da tag video, substituindo a tag de vídeo pela tag audio, que trabalhará com o áudio. Segue exemplo:

                    <audio controls>
                        <source src="musica.ogg" type="audio/ogg">
                        <source src="musica.mp3" type="audio/mpeg">
                        Seu navegador não possui suporte para áudio.
                    </audio>

Uma tag diferente e curiosa

Para você que gosta de brincar com o comportamento das páginas ou até mesmo necessita abrir uma página, mas não quer sair da página atual e nem gerar uma aba ou janela nova no navegador, exite a tag iframe. Esta tag abre uma nova página dentro da página do navegador. Atualmente ela é pouco usada, mas caso seja necessário, aí está um exemplo de aplicação dela:

As tags de formulário

voltar

As tags de formulário são muito utilizadas para obter informações do usuário, realizar cadastros, receber opiniões, entre outros. São importantíssimas para qualquer ramo do mercado.

Para iniciar um formulário, incluimos a tag form. Posteriormente, devemos incluir o conteúdo do formulário. Digamos que, por exemplo, você queira incluir três campos, sendo dois para coletar informações e um para receber uma mensagem. Desta forma, para criar os campos de preenchimento, deverá utilizar a tag input, enquanto para o campo referente à mensagem, deverá utilizar a tag textarea.

A tag input

A tag input possui o atributo type, que varia entre diversos tipos. Também há o atributo placeholder, que é um texto que ficará disponível enquanto nada for digitado nesse campo. Também é importante definir um atributo name para cada input. Alguns exemplos de tipos de input são:

  • type=”text”: Define um campo que receberá qualquer caractere
  • type=”email”: Define um campo que receberá caracteres e verificará se o mesmo consiste em um e-mail válido
  • type=”submit” value=”ENVIAR”: Define um botão que servirá para o envio do formulário. Dentro dele, podemos atribuir o value, que será o texto dentro do botão de envio.

A tag textarea

Assim como a tag input, essa tag define um campo para o formulário. Porém, diferentemente, ela tem como principal característica ser uma área de preenchimento de texto, ou seja, permite que o usuário escreva um texto ou uma mensagem no seu interior. Também traz opções para que o usuário redimensione seu tamanho (resize). Dessa forma, podemos incluir uma area de texto utilizando a tag textarea.

Um exemplo de trabalho de formulário é:

                <form method="post">
                    <p>Digite Seu nome <p>
                    <input type="text" name="nome" placeholder="Digite seu Nome">
                    <p>Digite Seu e-mail<p>
                    <input type="email" name="email" placeholder="Digite seu E-mail">
                    <p>Digite sua sugestão para nossa evolução <p>
                    <textarea type="text" name="email" cols="22" rows="10" placeholder="Digite sua sugestão"> </textarea>
                    

                    <input type="submit" name="enviar" value="Enviar">
                </form>

As tags de estilos e scripts

voltar

Por fim, existe a tag style, que trabalha com CSS, e a tag script que trabalha com códigos em javascript.

A tag style deve ser incluída no head do seu código HTML. Dentro dessa tag, é possível incluir todo o seu código CSS, ou seja, seu código de estilos.

Já a tag script tem como objetivo incluir códigos de scripts ao seu HTML, podendo ser incluída em qualquer parte. Contudo, recomenda-se fortemente que seja inserida após a tag footer(o rodapé). Dessa forma, podemos incluir nela um código javascript.
